YTBLND

YouTube Blend - Collaborative Feed Generator

A desktop application that blends YouTube playlist and watch history data from multiple users to generate a shared recommendation feed with real-time chat capabilities.

About This Project

YTBLND is a sophisticated C++ desktop application designed for collaborative video discovery. It empowers groups of 2-8 users to blend their YouTube preferences into a unified, personalized recommendation feed. Users can upload their YouTube data via Google Takeout exports, and the application automatically enriches metadata using the YouTube Data API v3. Beyond content discovery, YTBLND includes built-in chatrooms for blend-specific discussions, making it a complete social video exploration platform.

Core Features

User Management

Secure registration and login system with persistent user profiles and preferences.

Data Import

Upload YouTube data via Google Takeout CSV or HTML watch history exports seamlessly.

Metadata Enrichment

Automatic enhancement of video metadata including titles, channels, thumbnails, and logos via YouTube Data API.

Blend Creation

Create blends with 2-8 participants and customize your collaborative viewing experience.

Randomized Feed

Experience blended video recommendations in a dynamic 3×2 grid with intelligent randomization algorithms.

Real-Time Chat

Discuss videos with blend participants in dedicated blend-specific chatrooms.

Feed Navigation

Paginate through results with intuitive refresh functionality to discover more blended content.

Persistent Storage

Server-backed data persistence with SQLite support for local development and testing.

Technical Details

Technology Stack

  • Language: C++ (C++20 standard)
  • GUI Framework: wxWidgets 3.2+
  • Database: SQLite3
  • Networking: libcurl
  • APIs: YouTube Data API v3
  • Testing: GoogleTest
  • HTML Parsing: Gumbo Parser
  • Build System: CMake 3.16+

Architecture

  • Modular Design: Separation of UI, business logic, data parsing, algorithms, and persistence
  • Data Layer: Handles YouTube CSV and HTML export parsing
  • Algorithm Layer: Implements blending and randomization logic
  • Backend Integration: Server-backed persistence with local SQLite fallback
  • API Integration: Real-time metadata enrichment from YouTube

System Requirements

  • C++20 compatible compiler (GCC or Clang)
  • CMake 3.16 or later
  • wxWidgets 3.2+
  • libcurl for HTTP requests
  • SQLite3 (optional for local persistence)
  • Gumbo Parser (for HTML parsing support)

Build & Installation

Dependencies

Arch Linux:

sudo pacman -S cmake gcc wxwidgets-gtk3 curl sqlite gumbo-parser gtest

Ubuntu / Debian:

sudo apt install cmake g++ libwxgtk3.2-dev libcurl4-openssl-dev libsqlite3-dev libgumbo-dev libgtest-dev

Fedora / RHEL / CentOS:

sudo dnf install cmake gcc gcc-c++ wxGTK3-devel libcurl-devel sqlite-devel gumbo-parser gumbo-parser-devel gtest-devel

macOS:

brew install cmake wxwidgets curl sqlite gumbo-parser googletest

Building from Source

cd YTBLND-group45_cs3305proj
./build_src.sh

Running Tests

./run_tests.sh
./test_live_backend_connectivity.sh
