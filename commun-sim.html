Commun-Sim

Community Simulator Game

Explore the positive effects of community-focused funding and methods of providing safety through interactive simulation.

About This Project

Commun-Sim is an educational simulation game where players manage community resources and respond to neighborhood events. Experience how different community interventions impact neighborhood stability, safety, and quality of life. The game features a dynamic simulation loop with various building types, agents, and households that interact to create emergent gameplay around community wellbeing. To read more on the background for Commun-Sim and the research that inspired it, visit here.

Core Features

Dynamic Simulation

Real-time simulation loop where homes emit events based on stability. Watch as your decisions ripple through the community.

Building Management

Construct and manage Police Stations, Crisis Response Hubs, Social Services Offices, and Community Centers strategically.

Agent System

Specialized agents (Police, Crisis Responders, Social Workers, Community Workers) respond intelligently to neighborhood needs.

Event-Driven gameplay

Conflicts, mental health crises, and community needs occur dynamically based on neighborhood conditions.

Resource Management

Manage budgets, build infrastructure, and allocate resources to improve stability and safety metrics.

Interactive UI

Intuitive interface with build menus, statistics panels, event logs, and budget displays.

Technical Details

Technology Stack

  • Engine: Godot Engine 4.6+
  • Language: GDScript
  • Platform: Windows, macOS, Linux
  • Architecture: Multi-node scene-based design

Key Components

  • Sim_Manager: Core simulation autoload
  • BuildingManager: Building placement & operations
  • AgentManager: Agent behavior & spawning
  • EventManager: Event generation & distribution

Requirements

  • Godot Engine 4.6 or later

Get Started

Quick Start

  1. Download and install Godot Engine 4.6
  2. Clone the repository: git clone https://github.com/slipnut44/Commun-Sim.git
  3. Open Godot and load the src_ folder
  4. Click Play (F5) to run the game in the editor

Building an Executable

  1. In Godot, go to File → Export Project
  2. Create or select an export preset for your platform
  3. Configure export settings as needed
  4. Click "Export Project" and select an output directory
  5. Run the generated executable

Using the Prebuilt Binaries

  1. Clone the repository with git clone https://github.com/slipnut44/Commun-Sim.git
  2. Or, navigate to the repo. Then navigate to src_/
  3. Download the zip file titled sample_files.zip and extract it. Ensuring that the extracted file contents are kept within the same directory.
